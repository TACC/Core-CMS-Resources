MPI Enabled Libraries

There are 13 of these that do I/O, FFT signal analysis, and solvers.

Numerical Libraries
Twenty-two scientific computing routine collections form the basic foundation of Frontera’s user community. I/O supported by HDF5, NetCDF; Fast Fourier Transform analysis by FFTW2, FFTW3, MKL; Solvers by Blis, Elemental, Eigen, gsl, MKL, SuperLU.
MPI Libraries
Intel MPI 18.0.5 and 19.1.1 compilers scale to over 250k tasks, supports tuning collectives, detailed runtime and profile information. MVAPICH2 2.3.2 initializes quickly, scales to over 250k tasks, and also supports tuning.
